# Code and geophysical data for `Strong Ku-band altimetric backscatter from flooding on Antarctic sea ice'

This respository contains analysis code for a paper submission.

KuKa data from the Resolute, Churchill, MOSAiC and Rothera campaigns can be found at \url{https://doi.org/10.34992/ep9a-xt58}, \url{https://doi.org/10.34992/11tj-wr82}, \url{https://doi.org/10.5285/0caf5c54-9a40-4a96-a39b-b5c9c2863271} and \url{https://doi.org/10.34992/pqvf-q516} respectively.
